# Budget Flow - Personal Finance Management Application

Budget Flow is a comprehensive financial management application designed to help users track, manage, and optimize their personal finances. Built with React frontend and Spring Boot backend, this application provides a seamless experience for managing your financial life.

## Key Features

- 📊 **Dashboard Overview**: Get a quick snapshot of your financial health with visual summaries of income, expenses, and savings
- 👛 **Multi-Wallet Management**: Create and manage multiple wallets with different currencies and purposes
- 💸 **Transaction Tracking**: Record and categorize all your financial transactions with detailed filtering and reporting
- 👨‍👩‍👧‍👦 **Shared Wallets**: Collaborate on finances by sharing wallets with family members or partners
- 📈 **Financial Insights**: Visualize your spending patterns with interactive charts and graphs
- 🏷️ **Category Management**: Organize transactions with customizable categories for better financial tracking
- 🔔 **Notifications System**: Stay informed about important financial activities with real-time notifications
- 👤 **User Profiles**: Personalize your experience with customizable user profiles
- 🌐 **Multi-language Support**: Application supports internationalization for a global user base
- 📱 **Responsive Design**: Access your finances from any device with a fully responsive interface
- 📑 **Excel Export**: Export transaction data to Excel for detailed analysis and reporting

## Technology Stack

### Frontend
- React with Material UI for a modern, responsive interface
- Vite for fast development and optimized builds
- React Router for navigation
- i18next for internationalization
- Chart.js for financial data visualization

### Backend
- Spring Boot for robust API development
- Spring Security with JWT for authentication
- MySQL database for data persistence
- JPA/Hibernate for object-relational mapping
- Caffeine for efficient caching
